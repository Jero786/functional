# Functional

An attempt to create a simple functional programming library with monads.

This library started with Douglas Crockford's code used in his talk "Monads and Gonads"
<https://www.youtube.com/watch?v=b0EF0VTs9Dc>

## monads
* Basic monad
* Identity monad
* Maybe monad
* Either monad
* Promise monad
* State monad
